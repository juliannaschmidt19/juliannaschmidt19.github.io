Blocface

The type face that I've created is called Blocface, which is a mix between block letters and geometric shapes. It was made by connecting different shapes together with the addition of abstract cut outs and pop outs within each letter. For example, the "R" shape is made with rectangles of different shapes that create an abstract form a normal looking letter R. Each letter also has a dynamic interaction within each different shape. For example, with the letter R, if you hover over each individual shape, it turns it to a different color. This type face could be used for computer or video game design as well as in graphic design.
